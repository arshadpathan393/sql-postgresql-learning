# Database Setup

This folder contains scripts required to create the classic Oracle EMP, DEPT, and SALGRADE dataset in PostgreSQL.

## Execution Order

1. dept.sql
2. salgrade.sql
3. emp.sql
4. verify_setup.sql

## Tables

### DEPT
Department master table.

### EMP
Employee table containing 14 sample records.

### SALGRADE
Salary grade lookup table.

## Purpose

These tables are used throughout the repository to practice:
- SELECT
- WHERE
- GROUP BY
- HAVING
- JOINS
- Subqueries
- CTEs
- Window Functions
